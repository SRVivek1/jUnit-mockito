Topics
---------------
This project demonstrates the use of below mention jUnit annotations.

	1. assertArrayEquals(expectedResult, testData);
		>> Compare data of 2 arrays.
		
	2. @Test(expected = AssertionError.class)
		>> Helps to run negative testing.
		
	3. @Test(expected = NullPointerException.class)
		>> Helps to test expected Exceptions.
		
	4. @Test(timeout = 60)
		>> Helps to test and maintain performance of a method.
